User Manual

1. Enter The Interface

Enter the interface by clicking on enter dashboard or launch engine.

2. Target Definition

Add a 'Target Stream' (e.g., Maths), select your difficulty (Beg/Int/Adv), and hit 'Append Target'.

3. Execute

Click 'Request AI Blueprint'. The engine will map out your logic and start the focus timer automatically. You can later refine, regenerate, and customize and download your blueprint as needed. Enter your API key if shared key is not Available.
